Skills Acquired in Computer Systems & Architecture

Embedded system

An embedded system is a specialized computer designed to perform dedicated tasks within a larger system. Unlike general-purpose computers, embedded systems are built to manage specific functions efficiently. In the context of TVET education and Bigogwe TSS, students learn to maintain, repair, and program embedded systems used in devices like: Telephones and smartphones Computers and laptops Home appliances (e.g., microwaves, smart TVs) Industrial machines and robotics

Maintenance

Maintenance of devices like telephones, computers, and other embedded systems.

Enclosure Handling

Proper assembly, disassembly, and maintenance of computer and electronic enclosures.

LCD Screen Setup

Installation, configuration, and troubleshooting of LCD screens for computers and devices.

Programming

Basic programming skills for developing software and controlling hardware systems.

Main Components of Computer Architecture

  • CPU (Central Processing Unit): The brain of the computer that processes instructions.
  • Memory: Stores data and instructions for processing.
  • Input Devices: Devices like keyboard and mouse to interact with computers.
  • Output Devices: Monitor, printer, and speakers for information display.
  • Storage: Permanent data storage using HDDs, SSDs, or external drives.
